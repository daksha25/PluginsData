open git bash


git clone https://code.cognizant.com/514740/jenkins-plugins.git

goto root folder

git status
git add -A
git commit "msg"

Now secnd time want to commit again 
first need to pull that thing --> goto root directory where .git must be present

git pull origin master
will pull ur code make changes
